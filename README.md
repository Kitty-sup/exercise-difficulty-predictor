# Exercise Difficulty Predictor

A Machine Learning web application built using Python, Scikit-learn, and Streamlit that predicts exercise difficulty levels based on exercise-related features.

## Features

* Exercise difficulty prediction
* Data preprocessing pipeline
* Interactive Streamlit UI
* Logistic Regression model

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* Joblib

## Model Workflow

Dataset → Preprocessing → Encoding → Logistic Regression → Prediction

## How to Run

```bash
streamlit run app.py
```
